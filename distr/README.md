# Папка исходных файлов

Эта папка предназначена для файлов, которые пользователь передает ИИ в работу.

Правила для ИИ:

- файлы в этой папке считаются исходниками;
- читать их можно только как входные данные;
- изменять, перезаписывать, форматировать, удалять или переносить эти файлы нельзя;
- результаты обработки нужно сохранять в рабочие файлы проекта вне `distr`.
